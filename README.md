# Bosnia-Simulator-gam

Уважаемые игроки Bosnia Simulator.
В игру внедренна система «недостоющих файлов». Что это такое?
Это такая система, если у вас нету к примеру папки images то вам вылетит ошибка и вас направит на этот сайт.

Также игра абсолютно бесплатная и свободная для изменений.
Не защищена авторскими правами. Пользуйтесь свободно, добавляйте моды, меняйте код (он в исходном положении, то есть вы можете менять его)
Добавляйте хоть игру-копирку просто с другим названием. На вас не будет иска, ничего не будет вам.

Сразу хочу упомянуть, что создатель оригинальной игры на гитхабе - https://github.com/grationz720-dotcom .
Я буду очень рад если вы будете играть в эту игру.

ИНСТРУКЦИЯ ДЛЯ ИЗМЕНЕНИЯ И КОДОВЫЕ ВОЗМОЖНОСТИ

Эта фотка - расположение файлов в папке «Bosnia_Simulator», папке, где все основное.
Файл с расширением .spec - специальный файл созданный при сборке запускового файла .exe.
Файл .exe открывает игру - просто тыкните 2 раза по нему и откроется окошко pygame с игрой.

<img width="659" height="172" alt="image" src="https://github.com/user-attachments/assets/5f60cfa2-be57-44a2-8811-59b9a0765de9" />
Вот мы перешли в самое ядро игры.
В папке «sounds» основные звуки.
В папке «fonts» шрифты.
В папке «scripts» есть main.py, это и есть исходный файл с кодом. Без него игра не будет работать.
В папке «build» системные файлы при сборке файлов кода Python в .exe. Не удалять!

Начальное название файлов:
Images: icon.ico, player.png, mine.png, max.png, zhirinovsky.png. Если вы хотите поменять картинки игры то сохраняйте их в таких названиях (разрешайте заменить файл). Иначе у вас просто вылетит игра из-за недостатка файла.
Scripts: main.py, main.spec. mine.spec лучше вообще не трогать, а mine.py - исходный код игры который вы можете изменять. Главное не забывайте нажимать ctrl + s для сохранения!
Sounds: artiljerija.mp3, boom.mp3. Пока файл boom.mp3 не нужен (в официальной версии игры), а artiljerija.mp3 это фоновая музыка. Если вы хотите поменять звуки игры то сохраняйте их в таких названиях (разрешайте заменить файл). Иначе у вас просто вылетит игра из-за недостатка файла.

Также не стоит удалять другие папки. Изменять название файлов (если вы не поменяли код для них), создавать другие папки можно.
